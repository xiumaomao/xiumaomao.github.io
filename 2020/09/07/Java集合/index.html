Java集合

发表于 2020-09-07 | 分类于 集合
首先我们看上面的那幅图我们可以知道,所有的集合类都位于java.util包下,Java的集合主要有两个接口派生出来,分别是Collection和Map。 Collection下又分为List和Set,我们先来说一下list.

List

List下分为ArrayList,Vector和LinkedList

ArrayList

ArrayList底层是基于数组实现的,查询比较快,增删慢,他是线程不安全的,不能用于并发情况下使用.

  • add()方法 新增一个元素 返回值是一个boolean,新增成功返回true,新增失败返回false

    控制台输出:a b 控制台输出: a b a

  • remove(元素名称)方法 删除一个元素 返回值是一个boolean,成功返回true,失败返回false

    控制台输出:b a

  • remove(索引值,0开始)

    输出结果:a a

  • get(索引值)根据索引获取指定元素

    输出结果:b

  • clear() 清空集合所有元素

  • removeAll(集合) 清空所有元素

  • contains(元素名称) 集合中是否包含某个元素,包含返回true,不包含返回false

  • toArray() 将List集合转换数组

LinkedList

底层数据结构是链表,查询慢,增删快,他不是线程安全的,不能在并发条件下使用

  • add()方法 新增一个元素 返回值是一个boolean,新增成功返回true,新增失败返回false

    控制台输出:a b

  • remove(元素名称)方法 删除一个元素 返回值是一个boolean,成功返回true,失败返回false

    控制台输出:b a

  • remove(索引值,0开始)

    输出结果:a a

  • get(索引值)根据索引获取指定元素

    输出结果:b

  • clear() 清空集合所有元素

  • removeAll(集合) 清空所有元素

Vector

它的底层是数组实现的,查询快,增删慢;他是线程安全的,所以效率很低,它的所有方法在底层都添加了Synchronized关键字,给所有方法都加了锁。

  • add()方法 新增一个元素 返回值是一个boolean,新增成功返回true,新增失败返回false

    控制台输出:a b

  • remove(元素名称)方法 删除一个元素 返回值是一个boolean,成功返回true,失败返回false

    控制台输出:b a

  • remove(索引值,0开始)

    输出结果:a a

  • get(索引值)根据索引获取指定元素

    输出结果:b

  • clear() 清空集合所有元素

  • removeAll(集合) 清空所有元素

List集合允许元素重复,元素是有序的,可以使用普通for打印,有索引值

Set

Set分为HashSet,TreeSet和LinkedHashSet

HashSet

它的底层数据结构是hash表(无序,唯一)

  • add()方法 新增一个元素 返回值是一个boolean,新增成功返回true,新增失败返回false

    控制台输出:a b 当我们添加两个相同的数据时, 控制台输出: false a b

  • remove(元素名称)方法 删除一个元素 返回值是一个boolean,成功返回true,失败返回false

    控制台输出:b c

  • toArray() 将set集合转换成数组

  • clear() 清空集合所有元素

  • removeAll(集合) 清空所有元素

LinkedHashSet

它的底层数据结构是链表和哈希表
1.由链表保证数据有序
2.由哈希表保证数据唯一

TreeSet

它的底层数据结构是红黑树

Set集合不允许元素重复,元素无序并且没有索引

Map

Map接口是自成一个体系,他不是collection体系下的,它又可以分为HashMap,HashTable和TreeMap

HashMap

HashMap存储key-value键值对

HashMap和TreeMap不是线程安全的,HashTable是线程安全的。

List和Set区别

1.list是有序的,允许元素重复;set是无序的,不允许元素重复
2.list有索引,可以使用普通for循环遍历;set没有索引

ArrayList,Vector,LinkedList

1.ArrayList是基于数组实现的,不是线程安全的,查询快,增删慢
2.LinkedList是基于链表实现的,不是线程安全额,增删快,查询慢
3.Vector是基于数组实现的,是线程安全的,效率很低

HashSet,TreeSet,LinkedHashSet

1.HashSet它的底层是hash表实现的,主要功能是存储数据
2.TreeSet它的底层是红黑树,主要功能用于排序
3.LinkedHashSet它的底层是链表和hash表,主要功能用于保证FIFO即有序的集合(先进先出)

Collections

Collections是一个包装类,它包含了各种有关集合操作的的静态方法,此类不能被实例化,就像一个工具类,服务于Java的Collection框架。
