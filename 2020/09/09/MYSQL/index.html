MYSQL

发表于 2020-09-09 | 分类于 MYSQL

首先市面上那么多数据库软件,为什么学习MYSQL,我们先来看一幅图

这幅图可以看出MYSQL现在还是非常流行的,并且许多公司开发都是使用MYSQL数据库,接下来我们一起学习一下MYSQL数据库。安装我就不说了,网上比比皆是。

MYSQL常见命令

  • show databases;显示已经创建的数据库

    此时你应该会疑惑,我没有创建这4个数据库啊?其实这三个数据库是系统自动帮我们创建好的。 information_schema:保存数据库中数据的信息 mysql:保存数据库中用户的信息 performance_schema:保存数据库性能信息和参数 sys:保存系统测试信息

  • use 数据库名称:指定我们操作的数据库名称(数据库表是建立在数据库之上的,所以在操作表之前我们需要先确定操作的数据库)

  • show tables:显示该数据库下所有的表

  • show tables from 数据库名称:显示指定数据库下的数据库表(和上面的区别就是你不需要先指定数据库)

  • select database():查看自己所在的数据库

  • create table(。。。):创建数据库表并指定字段

  • desc table:查看数据库表的结构

  • select version():查看数据库版本

MYSQL语法规范

1.不区分大小写,但建议关键字大写,别名,表名小写
2.每条命令最好用分号结尾
3.注释 单行注释:# 多行注释: –或者/* */

基础查询

  • select 查询列表(表中字段,常量值,表达式,函数) from 表名

  • 查询表达式

  • 查询函数

  • 别名(1.用as关键字2.空格)

  • 去重distinct

    上图查出来的id有很多重复的,我们使用distinct关键字去重复

  • ‘+’作用
    在mysql中+号的作用仅仅只有运算符的作用,如果有一个是字符串,会首先把它解析成数字,如果解析成功就执行运算,如果解析失败,就将字符型数值转换成0.只要一方是null,结果肯定是null

条件查询

  • select 查询列表 from 表民 where 条件

  • between…and

模糊查询like(%:任意多个字符;_一个字符)

查询员工名中包含字符a的员工

函数

字符函数

  • length 返回字符长度

  • concat:拼接字符

  • upper,lower(将字符转换为大小写)

  • substr,substring:截取字符(索引从1开始)
    substr(‘字符’,start):表示从指定位置开始到结束

    substr('字符',start,len):表示从指定位置开始,指定截取长度

  • trim:去掉前后空格

  • replace:替换字符

数学函数

  • round:四舍五入

  • round(有参数):保留小数位

  • ceil:向上取整

  • floor:向下取整

  • truncate

  • sum:求和

  • count:计算个数

日期函数

  • now():获取系统当前日期+时间

  • curdate():获取系统当前日期(不包含时间)

  • curtime():获取当前时间(不包含日期)

  • year,month,day:获取指定日期的年,月,日

  • date_format:将日期转换成字符

分组查询(group by)

格式:select * from … where … group by … having … order by … limit
案例1:查询每个工种的最高工资

案例2:查询每个位置上的部门个数

案例3:查询邮箱中包含a字符的,每个部门的平均工资

案例4:查询每个部门的员工个数>2

having和where过滤条件的区别就是where是在表中就可以判断,having是对分组之后的结果进行过滤。

联表查询(join)

  • left join(左连接):以左表为主表,可以查询出左表的所有记录,不一定和右表匹配

  • right join(右连接):以右表为主表,可以查询出右表的所有记录,不一定和左表匹配
