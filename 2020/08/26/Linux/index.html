基本命令

文件处理命令

ls(显示目录文件)

-a 显示所有文件,包括隐藏文件

-l 详细信息显示(包括文件大小,所属者)


前面开头是d表示是一个目录,-开头表示是一个文件,l开头表示是一个软连接
文件权限:

权限分为三个部分,每个部分对应三个字母,r表示读权限,w表示写权限,x表示可执行文件
前三个字母代表的是u(所有者),第二个三个字母代表的是g(所属组),最后三个字母代表的是o(其他人)
例如:-rw——- 这个表示的就是所有者拥有读写权限,所属组的没有权限,其他人也没有权限
-lh 可以详细显示文件大小(自动选择单位)
-d 查看目录属性(显示自身文件的属性)

mkdir
表示在tmp目录下创建maomao这个目录 -p 表示递归创建这个目录 如果创建的中间目录没创建,就例如上面那个例子,我想在m目录下创建n目录,但是m目录没有,所以就创建失败了。 这时候就可以使用-p命令进行递归创建,他会在tmp目录下先创建m目录,再在m目录下创建n目录
pwd(显示当前目录)
rmdir(删除目录)
  • 只能删除空目录 rmdir 目录名称
cp(复制命令)

cp 源文件 目标目录 例如:cp maomao.txt /tmp 表示将maomao.txt复制到tmp目录下
-r 源目录 目标目录 例如:cp /tmp/maomao /maomao 表示将tmp目录下maomao这个目录复制到root目录下
-p 可以保存复制文件的属性(例如文件的修改时间不会变还是之前的)

mv(剪切)

mv /maomao /tmp 表示将root目录下maomao这个目录剪切到tmp这个目录下
mv /maomao /tmp/maomao1 表示将maomao这个目录剪切到tmp目录下,并改名为maomao1
如果在同一个目录就是改名,mv maomao maomao1 表示将maomao改名为maomao1

rm(删除)

1.rm /tmp/a.txt 表示删除tmp目录下的a.txt
2.-f(表示强制删除,没有提示消息)
3.-r(表示删除一个目录)
一般常用 rm -rf 目录名称

touch(新建一个文件)

touch 文件名称
touch program files 表示会创建两个文件
touch “program files” 表示创建一个带空格的文件

cat(查看文件内容)

cat 文件名称
-n 显示行号

tac(查看文件内容倒着显示)
more(分页显示文件内容,不能往回翻页)

more 文件名称
空格(或f) 翻页
Enter 换行
q或Q 退出

less(分页显示文件内容,可向上翻页)

less 文件名称
空格(或f) 翻页
Enter 换行
q或Q 退出
上箭头 一行一行往上翻页
pageup 一页一页往上翻页
在浏览状态下,按下/，可以进行名称搜索,搜索到的内容反白显示,例如:

n 可以匹配下一个符合关键词的地方

head(显示文件前几行)默认前10行

-n 显示行数
例如: head -n 10 文件名称

tail(显示文件后面几行)默认10行

-n 显示行数
-f 动态显示文件末尾内容(一般用于日志文件)

权限管理命令

chmod

  • 第一种方式
    chmod u+x 文件名称 给文件所有者加上执行权限

    同时做多个授权 给文件所属组加上w权限,其他人减去r权限 =表示不管之前什么权限,现在我赋予它新的权限 我不管他之前什么权限,现在就赋予它rwx权限

  • 第二种方式
    权限的数字表示
    r — 4 w —-2 x —-1
    例如: rwxrw-r– 就是764

    上图,将文件权限改为rw-r----- 对应的数字就是640

-R 递归修改
chmod -R 777 /tmp/a 表示将a目录同时将a目录下的所有目录权限都改掉

文件权限
r:cat/more/less..可以查看文件内容
w:vim表示可以编辑修改文件
x:脚本,命令
目录权限
r:ls表示可以查看
w:touch/mkdir/rmdir/rm
x:cd

chown(改变文件或目录的所有者)

语法格式:chown root /tmp/a 表示将tmp目录下的a目录的所有者改为root

chgrp(改变文件或目录的所属组)

语法格式:chgrp maomao /tmp/a 表示将tmp/a的所属组改为maomao

文件搜索命令

find(文件搜索)

语法格式:find 搜索范围 匹配条件
-name 根据文件名称搜索,例如 find /tmp -name maomao 表示在tmp目录下寻找名称为maomao的文件
find是精确搜索,他只会搜索名称为maoamo,如果想要模糊搜索可以搭配,?来使用 例如文件名称包含maomao的,*maomao;如果包含maomao的一个字符 maomao？来表示
-i 忽略大小写
-size 根据大小来搜索 find / -size +(-)204800 表示在根目录下搜索文件大小大于(小于)100MB的文件 一个数据块是linux存储额单位 1个数据块=0.5k
-user 根据文件所有者来搜索 find / -user maomao
-group 根据文件所属组来搜索 find / -group maomao
-cmin 根据文件属性被修改 find / -cmin 5 表示根目录下5分钟内文件属性被修改过的文件
-amin 根据访问时间
-mmin 根据文件内容
-a 两个条件都满足 -o 两个条件满足任意一个
find / -size +163840 -a -size -204800 表示在根目录下查找文件大小大于80MB,小于100MB的文件

压缩解压命令

gzip(压缩文件为.gz)

语法: gzip 文件

gunzip(解压缩.gz的压缩文件)

语法:gunzip xx.gz

tar(解压缩)

-x 解包
-v 显示详细信息
-f 指定解压文件
-z 解压缩
语法: tar -zxvf xx.gz

zip(压缩文件或者目录)

语法格式 zip 【-r】 压缩后的文件名 文件或目录

unzip(解压.zip文件)

进程管理

ps(显示系统执行的进程)

-a 显示当前终端的所有进程信息
-u 以用户的形式显示进程信息
-x 显示后台进程运行的参数
ps -aux | grep mysql 查询当前mysql中的进程

kill(终止进程)

-9 强制终止
kill -9 pid(进程编号)
